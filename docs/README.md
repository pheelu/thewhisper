# Documentazione — The Whisper

I documenti di architettura vengono generati dal workflow di design e riconciliati
in un modello dati unico. Struttura prevista:

- `ARCHITETTURA.md` — architettura unificata, schema DB, contratto API, eventi realtime, point economy, piano di build
- `FOUNDATION.md` — convenzioni trasversali, entità condivise, auth/sessione, realtime hub
- `domini/` — design di dettaglio per ciascun dominio
